# 📝 斯基GPT - 项目文案

## 🎯 项目概述

**斯基GPT** 是一个基于人工智能的智能内容聚合与推送系统，通过先进的RSS聚合技术和多模型AI处理，为用户提供精准的技术资讯推送服务。

## 💻 技术架构

### 核心技术栈
- **边缘计算平台**: Cloudflare Workers - 全球分布式部署，超低延迟响应
- **AI引擎**: OpenRouter多模型路由 - 智能调用最优AI模型
- **数据存储**: Cloudflare KV - 全球同步的键值数据库
- **消息推送**: Telegram Bot API - 实时消息传递
- **内容管理**: Payload CMS集成 - 现代化无头CMS

### AI模型策略 (RSS内容处理专用)
- **内容筛选**: Grok 4.1 Fast - 高速内容相关性判断
- **摘要生成**: Claude 3.5 Haiku - 高质量双语摘要  
- **翻译处理**: Claude 3.5 Haiku - 精准多语言支持
- **备用降级**: Groq Llama 70B - 可靠性保障

注：Telegram个人内容直接发布，不经过AI处理

## 🔄 实现方式

### RSS智能聚合系统
1. **多源采集**: 聚合30+顶级AI技术源
   - OpenAI、Google AI、DeepMind、HuggingFace等官方博客
   - arXiv、MIT、Stanford等学术机构
   - TechCrunch、Wired等科技媒体

2. **时区智能轮换**: 
   - 美洲时段 (00:00 UTC / 08:00北京): 美洲科技公司和媒体
   - 欧洲时段 (04:00 UTC / 12:00北京): 欧洲AI研究机构
   - 亚洲时段 (07:00 UTC / 15:00北京): 亚洲科技公司和实验室  
   - 全球时段 (11:00 UTC / 19:00北京): 国际AI新闻和分析
   - 深度时段 (14:00 UTC / 22:00北京): 深度技术内容

3. **智能去重**: URL指纹匹配，30天TTL防重复

### AI驱动内容处理
1. **相关性筛选**: 
   - Grok 4.1 Fast进行高速预筛选
   - 宽松策略：涵盖AI/ML、数据科学、编程工具等
   - 智能关键词识别和语义理解

2. **内容增强**:
   - 生成500字中文摘要 + 200字英文摘要
   - 提取关键技术点和创新亮点
   - SEO优化的标题和描述生成

3. **质量控制**:
   - 多模型验证机制
   - 降级策略保证服务稳定性
   - 实时性能监控

### 自动推送机制
1. **Telegram频道推送**: 
   - 自动发布到 @sijigpt 频道
   - 格式化消息模板
   - 链接预览和标签系统

2. **双向集成**:
   - Telegram → Payload: 频道手动消息直接发布到CMS（防循环机制）
   - RSS → Telegram: AI筛选内容推送到频道
   - 完整的Webhook处理和同步删除机制

### 部署平台
- **Cloudflare Workers**: 边缘计算平台，全球150+数据中心
- **Cloudflare KV**: 分布式键值存储
- **GitHub**: 代码版本管理和CI/CD

### 开发工具
- **Wrangler CLI**: Cloudflare Workers开发和部署工具
- **Node.js**: JavaScript运行时环境
- **Git**: 版本控制系统

### 监控工具
- **内置健康检查**: 实时系统状态监控
- **Cloudflare Analytics**: 性能和使用分析
- **自定义日志系统**: 详细的操作日志记录

## 🤖 AI模型选择 (仅用于RSS内容处理)

### 主力模型组合
1. **Grok 4.1 Fast (x-ai/grok-2-1212)**
   - 优势: 超高速度、理解能力强、成本效率高
   - 用途: RSS内容相关性筛选、实时判断
   - 特点: 长上下文支持、无地域限制

2. **Claude 3.5 Haiku (anthropic/claude-3-5-haiku)**
   - 优势: 优秀的指令遵循、多语言处理、稳定输出
   - 用途: RSS内容摘要生成、翻译处理
   - 特点: JSON格式输出稳定、理解力强

3. **Groq Llama 70B (groq/llama-3.1-70b-versatile)**
   - 优势: 高质量输出、开源模型、性价比高
   - 用途: 备用降级、复杂处理任务
   - 特点: 可靠性高、兼容性好

**注意**: Telegram频道的手动内容直接发布到Payload CMS，不经过AI模型处理，保持原始性和真实性。

### 成本优化策略
- 移除昂贵的Claude 3.5 Sonnet ($3/1M tokens)
- 优先使用高性价比模型组合
- 预计成本降低85-90%
- 智能降级机制保证服务质量
